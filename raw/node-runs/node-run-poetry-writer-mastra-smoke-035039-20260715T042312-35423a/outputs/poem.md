# 春雨初晴

## 七言绝句

云开雨霁晓风轻，  
碧瓦新沾宿雨明。  
一树杏花红欲破，  
春山如笑草如萌。

---

### 注释

| 项目 | 说明 |
|------|------|
| 体裁 | 七言绝句（平起首句入韵） |
| 韵脚 | 轻(qīng)、明(míng)、萌(méng) — 中华新韵·十一庚 |
| 主题 | 春雨初晴 — 描写雨后初晴的清新景象 |
| 意象 | 云开、雨霁、晓风、碧瓦、杏花、春山、草萌 |
| 情感 | 明快清新，表达春雨过后万物复苏的喜悦 |

### 赏析

首句「云开雨霁晓风轻」勾画雨后天开、晨风轻拂的整体氛围；  
次句「碧瓦新沾宿雨明」以碧瓦上的雨珠反衬天光，细节传神；  
第三句「一树杏花红欲破」以杏花含苞欲放的「红欲破」突出春意之浓烈；  
末句「春山如笑草如萌」以拟人手法收束全篇，点出山川含笑、草木萌发之生机。  
四句层层递进，从远到近、由景入情，完整呈现「春雨初晴」的意境。
